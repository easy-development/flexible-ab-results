Flexible AB Testing is an easy to use plugin that will help you handle your A/B Testing in less than a few minutes.

== Description ==

Flexible A/B Testing is an intuitive way to start doing your own A/B Testing in less than a few minutes.
All you need to do is install the plugin and create your first campaign.

== Installation ==

1. Upload `flexible-ab-results` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find the Menu page on the left "Flexible AB Results"
1. Create your campaign
1. Enjoy.

== Frequently Asked Questions ==

= Does this plugin work with external links / websites =

Yes it does, it's made to work out of the box, flexible, and easily.
